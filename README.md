# ENTREGA FINAL - DESAFIO QA SWAGLABS

## 🇧🇷 Descrição
Projeto de Qualidade de Software (QA) desenvolvido para o desafio da DIO, utilizando Jira, Zephyr e XMind para gestão de requisitos, planejamento, execução e evidências de testes no fluxo do SWAGLABS. Inclui documentação detalhada em PDF, mapa mental do checkout e evidências das principais histórias de usuário (login e checkout).

## 🇺🇸 Description
Software Quality Assurance (QA) project developed for the DIO challenge, using Jira, Zephyr, and XMind for requirements management, planning, test execution, and evidence collection in the SWAGLABS workflow. Includes detailed documentation in PDF, a mind map for the checkout process, and evidence of the main user stories (login and checkout).

## 📁 Entregáveis / Deliverables
- ENTREGA_DESAFIO_QA_SWAGLABS.pdf: Documentação principal / Main documentation
- MINDMAP_US02_CHECKOUT.pdf: Mapa mental do checkout / Checkout mind map
- DES-1.pdf: Evidência da história de login / Login user story evidence
- DES-4.pdf: Evidência da história de checkout / Checkout user story evidence

## 🚀 Como usar / How to use
1. Abra os arquivos PDF para consultar o planejamento, execução e evidências dos testes.
2. Utilize o mapa mental para visualizar o fluxo do checkout.

1. Open the PDF files to review the test planning, execution, and evidence.
2. Use the mind map to visualize the checkout flow.
