# FINAL DELIVERY - SWAGLABS QA CHALLENGE

## Description
This Software Quality Assurance (QA) project was developed for the DIO challenge, using Jira, Zephyr, and XMind for requirements management, planning, test execution, and evidence collection in the SWAGLABS workflow. The repository includes detailed documentation in PDF format, a mind map for the checkout process, and evidence for the main user stories (login and checkout).

## Deliverables
- ENTREGA_DESAFIO_QA_SWAGLABS.pdf: Main documentation
- MINDMAP_US02_CHECKOUT.pdf: Checkout mind map
- DES-1.pdf: Login user story evidence
- DES-4.pdf: Checkout user story evidence

## How to Use
1. Open the PDF files to review the test planning, execution, and evidence.
2. Use the mind map to visualize the checkout flow.

## Tools Used
- Jira & Zephyr
- XMind
- Python (pypdf, reportlab)
- Git & GitHub
